# Customer Churn Prediction

This project was completed as part of the BCG Data Science Virtual Experience Program.

The main objective of this project was to analyze customer data and predict whether a customer is likely to churn or not using Machine Learning.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## What I Did

### Exploratory Data Analysis (EDA)

* Checked data types and dataset information
* Analyzed customer churn distribution
* Performed correlation analysis
* Created visualizations such as heatmaps, histograms and boxplots
* Compared energy prices for retained and churned customers

### Feature Engineering

Created some new features from the existing data:

* Customer tenure
* Renewal gap
* Monthly consumption
* Margin ratio
* Price difference between January and December

### Model Building

I used a Random Forest Classifier to predict customer churn.

The dataset was split into training and testing sets, and the model was evaluated using different performance metrics.

## Results

* Accuracy: 90.4%
* Precision: 80.7%
* Recall: 5.7%
* F1 Score: 11.0%

## Key Observations

* Customer consumption was one of the most important features.
* Margin-related features also had a significant impact.
* The dataset was imbalanced, which affected the model's recall score.
* The model performed well overall but can be improved further.
